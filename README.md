# homebrew-video-capture-mcp

Homebrew tap for `video-capture-mcp`.

The formula is generated by `.github/workflows/update-formula.yml` after a
`video-capture-mcp` release is published.

```bash
brew tap Corvus400/video-capture-mcp
brew install video-capture-mcp
```

## Security

Report security issues through the upstream repository's private security
advisory flow:
